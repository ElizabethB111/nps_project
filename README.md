# nps_project
NLP project classifying National Park Service alerts into traveler-centered categories using rule-based, TF-IDF, sentence-embedding, and transformer models, with a focus on generalization to unseen parks.

This repository contains an NLP project that classifies National Park Service alerts into traveler-centered categories based on how they affect visitor decisions. The project organizes alerts by practical impact, such as trail or area access, road and parking issues, environmental hazards, wildlife restrictions, facility or campground services, and general/minor notices. The workflow includes data collection from the official NPS API, manual annotation of a gold-standard dataset, baseline models, TF-IDF logistic regression, sentence-transformer embeddings, fine-tuned DistilBERT, held-out-park evaluation, grouped cross-validation, visualizations, and error analysis. The main goal is to test which NLP methods best generalize to alerts from parks that were not seen during training.
